# 爆款标题拆解标准

## 1. 选题
一句话说清真正主题与切口。
- 差：介绍公众号运营
- 好：30+ 女性转型做公众号的优势论——从“年龄焦虑”反向切入

## 2. 人群
写会被精准钉住的人，尽量包含年龄、身份、阶段、状态。

## 3. 痛点
写“为什么会点开”，不是复述标题。优先找深层焦虑、拧巴与具体场景。

## 4. 反常识观点
提炼它颠覆了什么默认认知。若没有，写“无”。

## 5. 情绪
用 1-3 个词概括核心情绪。常用词：焦虑、不甘心、羡慕、愤怒、认同、好奇、治愈、安全感、共鸣、优越感。

## 6. 传播点
判断它为什么会被主动传：
- 可截图表态的金句
- 可代入的身份
- 稀缺信息
- 可收藏清单 / 工具
- 能引发讨论或站队
- 适合转给“和我一样的人”

## 7. 钩子公式
把具体名词抽成 `[变量]`，保留真正承重的句式。
- 好：`[领域]最大的骗局，就是相信 [一条默认正确的常识]`
- 好：`你想成为 [高结果身份]，先破除这 [数字] 个 [内在阻碍]`
- 抽不出来：写“无明显公式”

## 8. 改写
直接迁移到“公众号写作获客”赛道。
改写时要同时迁移两层东西：
1. 句式骨架
2. 原标题真正的传播动能（如高结果诱惑、强身份代入、清单感、站队感、稀缺信息感）

如果两者冲突，优先保住传播动能，而不是只复刻句型。
- 贴图版：≤20 字，适合封面 / 图文
- 长文版：≤20 字，适合公众号正文标题
- 必须零修改可发
- 不要写“探讨 / 分析 / 介绍 / 如何看待”

## 参考示例

原标题：你想成为年入百万的作家，先破除这4个心魔

- 选题：讲“想靠写作赚到大钱，先要清掉哪些心理障碍”，从能力问题切到心态门槛。
- 人群：想靠写作变现、做个人品牌或成为职业作者的人；多半已经开始写，但收入还没明显起来。
- 痛点：她卡住的未必是不会写，而是迟迟不敢把写作当事业：怕被评价、怕卖不出去、怕自己不配赚大钱。
- 反常识观点：主流会把“年入百万作家”归因于文笔、流量、资源；这条先把原因压到“心魔”上，强调先破心理限制，再谈结果。
- 情绪：渴望 + 焦虑 + 自我怀疑
- 传播点：“年入百万”给了强结果诱惑，“4个心魔”又有清单感，适合收藏；同时会让很多写作者产生自我代入。
- 钩子公式：`你想成为 [高结果身份]，先破除这 [数字] 个 [内在阻碍]` —— 结果诱惑 + 自我诊断型骨架。
- 改写：
  - 贴图版：你想靠公众号月入5万，先破这4个心魔
  - 长文版：想靠公众号稳定成交，先过这4道心魔
- 说明：这里不只迁移了“你想成为……先破除……”的句式，也保留了原标题“高结果诱惑 + 清单感”的传播动能。
